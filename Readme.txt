**** 软件使用说明
比如我想通过该软件进行泰拉瑞亚IPv6联机
需要双方都下载该软件
服务器玩家在软件中设置 监听端口TCP6:9999 目标端口TCP4:7777 然后保存并运行即可，然后将服务器电脑的公网IPv6地址发送给其他玩家
其他玩家在软件中设置 监听端口TCP4:8888 目标端口TCP6:9999 目标主机填服务器电脑的公网IPv6地址，游戏内请使用127.0.0.1:8888地址

监听端口的端口号可以修改成任意空闲端口，目标端口号服务器玩家填游戏默认端口号（泰拉瑞亚的默认端口号是7777）、其他玩家填服务器的监听端口号

其他游戏也是同理
